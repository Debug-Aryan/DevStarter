## Getting Started

This is a React Native project using Expo.

### Prerequisites
- Node.js (v18+)
- Expo Go app on your phone OR Android Studio / Xcode for emulators.

### Installation
1.  Install dependencies:
    ```bash
    npm install
    ```

### Running the App
1.  Start the Expo dev server:
    ```bash
    npm start
    ```
2.  Scan the QR code with Expo Go (Android/iOS) or press `a` for Android Emulator / `i` for iOS Simulator.
