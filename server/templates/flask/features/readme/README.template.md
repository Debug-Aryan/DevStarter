## Getting Started

### Prerequisites
- Python 3.8+

### Setup
1. Create a virtual environment:
   \`\`\`bash
   python -m venv venv
   source venv/bin/activate  # Windows: venv\\Scripts\\activate
   \`\`\`

2. Install dependencies:
   \`\`\`bash
   pip install -r requirements.txt
   \`\`\`

3. Run the application:
   \`\`\`bash
   python run.py
   \`\`\`
