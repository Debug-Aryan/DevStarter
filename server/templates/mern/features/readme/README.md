## Getting Started

### Prerequisites
- Node.js (v18+)
- MongoDB

### Installation

1. Install dependencies
   ```bash
   cd server && npm install
   cd ../client && npm install
   ```

2. Environment
   - Copy `.env.example` to `.env` in root
   - Copy `client/.env.example` to `client/.env`

3. Run
   - Server: `cd server && npm run dev`
   - Client: `cd client && npm run dev`
