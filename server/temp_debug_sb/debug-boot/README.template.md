# debug-boot

Debug App

## Tech Stack
- Java 17
- Spring Boot 3
- REST API

## Getting Started

### Prerequisites
- JDK 17+
- Maven (or use ./mvnw wrapper)

### Build
\`\`\`bash
mvn clean package
\`\`\`

### Run
\`\`\`bash
java -jar target/*.jar
\`\`\`
